# Data Notes

This folder contains summary and demonstration datasets prepared from the published paper for GitHub documentation.

## Included files

### `benchmark_metrics.csv`
Comparison of the proposed LSTM-MHSA model against FSB, HDTFSSMM, LSTM, and MHSA benchmark models.

### `training_size_robustness.csv`
Robustness comparison for Case I, Case II, and Case III, corresponding to 20, 10, and 5 complete run-to-failure trajectories.

### `performance_gains.csv`
Performance gains of the proposed LSTM-MHSA model over LSTM and MHSA using the combined metric.

### `case_study_summary.csv`
Summary of the case study and dataset structure.

### `life_fractions.csv`
Life fractions used to evaluate model performance across the equipment life.

### `sample_rul_predictions.csv`
A small demonstration dataset for plotting RUL prediction curves. This is not raw experimental data.

## Data-sharing note

The full simulated trajectories are not included. The files are prepared for educational, research, and portfolio demonstration purposes based on reported values from the published paper.
